# Codex iOS Implementation Prompt

Use this prompt for P2 or later iOS work.

```text
You are working in the ai-app-starter-swiftui-fastapi repository.

Goal:
Implement the next iOS task for the current roadmap phase.

Rules:
- Stay within the requested phase.
- Do not configure App Store Connect or TestFlight.
- Do not create real secrets.
- Do not hardcode production URLs.
- Do not commit, push, deploy, or call external APIs without explicit confirmation.
- Keep the diff small and testable.

Before editing:
- Inspect ios/README.md.
- Inspect docs/ios-design.md.
- Inspect docs/auth-design.md if authentication is relevant.
- Inspect progress.md.

Expected output:
1. Files changed
2. Implementation summary
3. Tests or smoke checks
4. Commands run
5. Remaining risks
6. Suggested progress.md updates
```
